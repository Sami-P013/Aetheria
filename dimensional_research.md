## Platform Overview
- **Tagline**: "Personality testing that reveals your true self, compares you to your inner circle, and calls you out in the best way"

## Key Features Observed
- **Progressive Unlocking**: Users must complete certain assessments to unlock others
- **Status Tracking**: Shows completion progress (e.g., "2 of 15 Dimensions Complete", "17 of 203 Elements discovered")
- **Results Dashboard**: Multiple visualization types including trait circles, analysis cards, and comparison metrics
- **Premium Content**: Special reports like "Therapist Reference" and "You Report"
- **Social Integration**: Friend invites and comparisons
- **Mobile-First Design**: Clean, dark interface with card-based layout

## Visual Design Elements
- Dark theme with colorful accent elements
- Circular progress indicators for traits
- Card-based layout for different dimensions
- Icon system for each dimension type
- Color-coded trait visualization (pink, orange, yellow, green, purple, etc.)

## Assessment Flow
1. Start with Core Dimensions (Cognition, Values, Primary Traits)
2. Unlock Advanced Dimensions based on completion
3. Premium features require subscription
4. Results integrate across multiple dimensions for comprehensive analysis

## Unique Value Propositions
- Multi-dimensional approach (15+ dimensions vs typical single-dimension tests)
- Social comparison features
- Daily personalized insights
- Professional-grade reports for therapy/coaching
- Interconnected results showing relationships between different personality aspects

## Cognition Dimension - Jung-Based Assessment

### Theoretical Foundation
**Cognition measures how you think: how interpret reality, process information, and make decisions. The Cognition Dimension is based on Jung's model of Cognitive Functions.**

### Simplified Implementation
For simplicity's sake, function stacks have been replaced with a set of four opposite traits:
- **Introversion vs. Extroversion**
- **Sensing vs. Intuition** 
- **Thinking vs. Feeling**
- **Judging vs. Perceiving**
